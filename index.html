DjSuperAdminEdit contents directly on your page

A tiny, dependency-free Django app that turns any content on your templates into an inline, in-browser editor for superusers.

Quick Start →
View on GitHub

✍️ Edit in place

Superusers double-click any wrapped content and edit it right on the page — WYSIWYG or raw.

🪶 Zero dependencies

Nothing but Django. Compatible with every Django release from 2.0 to the latest.

🔌 Bring your own model

Use the built-in Content model, or wire the DjSuperAdminMixin to your own model and API endpoint.
